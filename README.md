  # Image Manager App
A GUI application that can view images from a folder and manages them.

# Features include:
- Rename image
- Delete image; moves image to the recycle bin
- Move image

 # Installation
```
pip install pillow
pip install pathlib
pip install shutil
pip install send2trash
```
