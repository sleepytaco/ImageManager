  # Image Manager
A python web scraping script that takes in a movie name as input and outputs various information about the movie.

# Features include:
- Rename image
- Delete files; moves image to the recycle bin
- Move image

 # Installation
```
pip install pillow
pip install pathlib
pip install shutil
pip install send2trash
```
